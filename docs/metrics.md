# Phase 1 Success Metrics and Pilot Definition

## Purpose
This document defines how the effectiveness and readiness of RapidMaternal will be evaluated during the Phase 1 pilot.

## Primary Metrics
- Time from emergency trigger to first acknowledgement
- Percentage of emergencies acknowledged within target time
- Number of escalations required per emergency
- System uptime during pilot period

## Secondary Metrics
- Number of emergencies triggered per facility
- Average response time by user role
- User adoption and engagement

## Pilot Scope
- Location: Zawse–Bansi Subdistrict, Binduri District
- Duration: 8–12 weeks
- Facilities: Zawse Health Centre and selected CHPS zones

## Go / No-Go Criteria
The system will proceed to wider deployment only if predefined performance, safety, and usability thresholds are met.
